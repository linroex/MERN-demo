# MERN Demo

[![Node.js CI](https://github.com/yubinTW/MERN-demo/actions/workflows/node.js.yml/badge.svg)](https://github.com/yubinTW/MERN-demo/actions/workflows/node.js.yml)

Mongo + Fastify + React + Node

## Build Frontend

```
cd frontend
npm i
npm run build
```

## Build Backend

```
cd backend
npm i
npm run build
npm run start
```
